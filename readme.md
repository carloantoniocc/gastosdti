instalar, usando la línea de comandos, nos dirigiremos a la carpeta var del
servidor</span></p>

<p class=Destacado><span lang=ES>cd /var</span></p>

<p class=MsoNormal><span lang=ES>una vez posicionados en la carpeta var,

</span></p>

<p class=Destacado><span lang=EN-US>git clone http://10.8.64.45/desarrollo/inicial.git</span></p>
